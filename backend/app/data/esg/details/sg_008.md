## 合规检查项
- [ ] 可持续报告是否设置短、中、长期目标
- [ ] 绩效数据是否与以前披露的目标进行对照
- [ ] 时间范围是否与战略规划和财务报告一致

## 评估标准   <!-- derived: true，需人工复核 -->
- 满足：设定明确时间目标和绩效对照
- 部分满足：有目标但缺乏绩效追踪
- 不满足：无目标设定

## SME 常见差距   <!-- derived: true -->
- IFRS S2 要求仅对实际排放值进行核查（verification），默认值不要求核查
- 目标应覆盖短、中、长期三个时间范围

## 法规原文摘要   <!-- 事实层：verbatim，引号标明 -->
> "The sustainability report should set out the issuer's targets for the forthcoming year in relation to each material ESG factor identified. Targets should be considered for defined short, medium and long term horizons, and if not consistent with those used for strategic planning and financial reporting, the reasons for the inconsistency should be disclosed."
> —— 来源：Practice Note 7.6, para 4.1(d)（英文原文）

# 智企通 MVP — 产品最终形态（v2）

## 一句话

一个网页应用。中小企业主打开浏览器，花 5-8 分钟走完 3 步，拿到一份**出海融资方案 + ESG 合规缺口报告**。

### v2 更新要点
- ESG 评估新增**双标准对照**（目的地法规 + BOCHK 准入）
- 新增**欧盟 CBAM** 作为第 4 个目标地区
- ESG 报告增加**可信度分层**（原文/AI 判断/AI 建议）
- 落地页增加**进度看板**

## 用户最终拿到什么

走完 3 步后获得四样东西：

| 产出 | 内容 | 用途 |
|------|------|------|
| **企业画像卡片** | 行业标签、规模等级、出海能力评估、雷达图 | 了解自身定位 |
| **融资产品推荐** | 匹配的 BOCHK 产品 + 匹配度 + 推荐理由 + 额度 | 知道该申请什么 |
| **ESG 缺口报告（目的地）** | 红黄绿交通灯 + 法规原文 + AI 判断/建议 + 置信度 | 知道目标国家合规差距 |
| **ESG 准入评估（中银）** 🆕 | 绿色贷款达标情况 + 达标路径 | 知道如何获得绿色融资 |

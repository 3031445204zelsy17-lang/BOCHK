# 智企通 UI 设计 Token — 基于 BOCHK 官网风格

> 来源：https://www.bochk.com 首页截图分析

## 配色

| 用途 | 色值 | 说明 |
|------|------|------|
| 品牌红（主色） | `#C8102E` | 按钮、强调、品牌标识 |
| 品牌红 hover | `#B01020` | 按钮悬停态 |
| 白色 | `#FFFFFF` | 背景、按钮文字 |
| 深灰（文字） | `#333333` | 正文、导航 |
| 浅灰（边框） | `#E0E0E0` | 卡片边框、分隔线 |
| 浅灰（输入框） | `#F5F5F5` | 输入框背景 |
| 浅蓝（辅助强调） | `#0071C5` | 标题、tab 选中 |
| 金色（点缀） | `#D4AF37` | 高端感点缀 |

## 排版

| 项目 | 值 |
|------|---|
| 字体 | 无衬线（系统字体栈：-apple-system, Helvetica, Arial） |
| 导航字号 | 14px |
| 标题字号 | 18px bold |
| 正文字号 | 12-14px regular |
| 导航字重 | 300-400 |
| 标题字重 | 700 |

## 组件

| 组件 | 规范 |
|------|------|
| 按钮 | 红底白字 `#C8102E`，圆角 4px，hover 变深 `#B01020` |
| 次要按钮 | 浅灰底 `#F5F5F5` + 深灰字 `#333333` |
| 卡片 | 白底，1px `#E0E0E0` 边框，圆角 4px，内边距 16px |
| 阴影 | 极少或无，偏扁平风格 |
| 导航栏 | 白底，水平排列，左 logo 右菜单 |
| 分隔线 | 1px `#E0E0E0` |

## ESG 报告专用色

| 状态 | 色值 | 说明 |
|------|------|------|
| 绿灯（满足） | `#22C55E` | Tailwind green-500 |
| 黄灯（部分满足） | `#EAB308` | Tailwind yellow-500 |
| 红灯（不满足） | `#EF4444` | Tailwind red-500 |
| 可信度-高 | `#3B82F6` | 蓝色 |
| 可信度-中 | `#F59E0B` | 黄色 |
| 可信度-低 | `#F97316` | 橙色 |

## Tailwind 自定义配置

```js
// tailwind.config.js 中的 theme.extend.colors
{
  bochk: {
    red: '#C8102E',
    'red-hover': '#B01020',
    gold: '#D4AF37',
    blue: '#0071C5',
    gray: {
      light: '#F5F5F5',
      border: '#E0E0E0',
      text: '#333333',
    }
  }
}
```
